Estilos de Texto
Descripción Propiedad Atributo ¿Cómo se veria?
Esta propiedad permite cambiar la fuente del texto. font-family Nombre de la fuente deseada. Ej: verdana Fuente de texto con verdana
Esta propiedad permite cambiar el color del texto. color Color de la fuente deseado. Ej: blue color azul del texto
Esta propiedad permite cambiar el tamaño del texto. font-size Tamaño deseado en px. Ej: 20px Tamaño de letra 20px
Esta propiedad permite cambiar el grosor de la fuente (negrita). font-weight Grosor deseado del texto. Ej: bold Texto en negrita
Esta propiedad permite definir si el texto estará en cursiva font-style italic para letra en Cursiva. Ej: italic Texto en cursiva
Esta propiedad permite alinear el texto text-align center para centrar, right para alinear a la derecha, left para alinear a la izquierda. Ej: center Texto a la derecha
Esta propiedad permite agregar subrayado o tachado al texto text-decoration underline para subrayar o line-through para tachar el texto. Ej: underline Texto subrayado
Esta propiedad permite agregar una sombra al texto text-shadow grosor en pixeles de la sombra del texto y el color de la sombra. Ej: 2px 2px 5px brown Texto con sombra
Estilos de Imagen
Descripción Propiedad Atributo ¿Cómo se veria?
Esta propiedad permite agregar un borde a la imagen border Borde en pixeles. Ej: 2px solid brown
Esta propiedad permite agregar un borde redondeado o circular a la imagen agregandole el overflow permite que no se visualicen las esquinas border-radius con overflow Porcentaje del borde y overflow con hidden para ocultar los bordes. Ej: 50%, hidden
Esta propiedad agrega una sombra a la imagen. box-shadow Valor en pixeles del grosor de la sombra, y el color de la sombra. Ej: 5px 5px 15px brown
Esta propiedad agrega filtros a la imagen. filter Tipo de filtro de la imagen. Ej: grayscale(100%)
Esta propiedad permite rotar o inclinar una imagen. transform Tipo de rotación que se quiere dar a la imagen. Ej: rotate(45deg)
